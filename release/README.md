# Release Layout

This folder contains the distribution-oriented FastLoop release scaffold.

- `out/`: versioned release artifacts generated by `npm run release:package`
- `templates/`: reserved for future installer, signing, and Adobe Exchange packaging templates
- `CHECKLIST.md`: release validation checklist before publishing a build

Current packaging output includes:

- a versioned CEP bundle
- the packaged Windows engine runtime when it has been built
- install and release notes copied into the release folder
- an unsigned zip artifact for staging

Recommended release flow:

```bash
npm run release:build
npm run release:validate
```

Signing and true installer generation remain the next follow-up step.
