# 2. Design Brief

## Design Intent

FastLoop should feel like a serious editor utility: compact, dark, efficient, and information-dense, with immediate access to important loop actions and no visual drift toward generic consumer audio software.

## Primary Visual Reference

Shutter Encoder is the core reference for:

- desktop utility density
- grouped control hierarchy
- sidebar-driven workflow organization
- immediate access to advanced options
- bottom batch queue readability
- dark neutral palette with functional accents

## Transformation Rules

FastLoop must preserve Shutter Encoder's workflow logic without copying brand, implementation, wording, or assets. The redesign should:

- keep the strong utility posture
- modernize spacing and typography rhythm
- prioritize waveform analysis in the center
- make loop candidates more visually comparable
- emphasize stateful scoring indicators
- improve selected-row and active-candidate legibility

## Visual System

### Tone

- practical
- editorial
- calm
- confident
- precise

### Density

- compact control rows
- reduced decorative padding
- tight inspector cards
- dense but readable queue rows

### Palette

- charcoal and graphite base
- muted steel borders
- cool blue for selection
- green for strong confidence
- amber for caution and click risk
- red only for blocking errors

### Typography

- condensed desktop-friendly scale
- strong distinction between labels, values, and statuses
- small caps or uppercase for utility headers only

## Key Screens

- library and analysis screen
- candidate ranking screen
- bed-builder screen
- export planning screen
- queue and preset management screen

## Motion

- restrained state fades
- handle hover emphasis on waveform
- subtle selected-row lift
- animated confidence bars only where useful

## UX Principles

1. Show the important controls up front.
2. Keep advanced scoring visible, not buried.
3. Make every score explainable.
4. Preserve batch mentality throughout the app.
5. Let the waveform stay central to decision-making.
